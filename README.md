This project demonstrates end-to-end MongoDB operations using Java.

## Features
- MongoDB connection using MongoClient
- Insert single & multiple documents
- Read with filters, projections, and sorting
- Update single and multiple documents
- Delete documents using regex filters
- Aggregation pipelines:
  - Count by role
  - Average age by role
  - Sum of age by role

## Tech Stack
- Java
- MongoDB
- MongoDB Java Driver
- Maven

## Use Case
Designed as a hands-on MongoDB project for QA Automation / SDET profiles to demonstrate backend data validation skills.
